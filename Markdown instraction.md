## Списки

Чтобы добавит ненумерованные списки нужно пункты выделять звездочкой (*). 
Например, вот так:
* Элемент 1
* Элемент 2
* Элемент 3

Чтобы добавить нумерованные списки надо пункты лишь пронумеровать. Например, вот так:
1. Первый пункт
2. Второй пункт
3. Третий пункт
